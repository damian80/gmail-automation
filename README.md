# Gmail Automation

Python tool that connects to Gmail API and uses Claude AI to automatically classify emails as IT support requests.

## Features
- OAuth 2.0 authentication with Gmail API
- Reads unread emails from inbox
- AI-powered classification using Anthropic Claude API
- Detects IT support requests automatically

## Tech
- Python
- Google Gmail API
- OAuth 2.0
- Anthropic Claude API
